# Celebrity Relations Graph

A Python tool to generate a relationship graph for a given celebrity by scraping web data and using an LLM to extract relationships.

## Features

- **Web Scraping**: Fetches celebrity information from Wikipedia.
- **Relationship Extraction**: Uses an LLM to identify relationships between people.
- **Visualization**: Generates an interactive network graph.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

```bash
python src/main.py --name "Celebrity Name"
```
